📘 About Project

Student Performance Prediction System Using Machine Learning

🎯 Project Objective

To predict student academic performance using Machine Learning techniques and provide meaningful insights through visual analytics and comparison dashboards.

🛠 Technologies Used

  • Python
  • Flask
  • Pandas
  • Scikit-Learn
  • Matplotlib
  • HTML
  • CSS

📂 Dataset Features

  • Attendance
  • Midterm Score
  • Final Score
  • Assignments
  • Quizzes
  • Projects
  • Study Hours
  • Stress Level
  • Sleep Hours

🤖 Machine Learning Model

Random Forest Classifier was used for predicting student grades. Feature importance analysis was performed to identify influential academic factors.
